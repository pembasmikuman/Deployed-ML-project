# How to start
### For Windows 11
1. Clone the repository using either `git clone https://github.com/pembasmikuman/Deployed-ML-project.git` or Download as `.zip`.

2. Open terminal and change directory to the repository folder using `cd .\Deployed-ML-project\`

3. Initialize virtual environment
`python venv -m .venv`

4. Activate the virtual environment using `.\venv\Scripts\Activate.ps1`

5. Install dependencies with `pip install streamlit tensorflow pandas numpy scikit-learn`

6. Finally, run the local web interface with `streamlit run app.py`
